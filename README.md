# Hansa AI Assistant

Hansa is a desktop AI assistant built using Python that can perform various tasks such as answering questions like ChatGPT, opening desktop applications, browsing websites, and even making phone calls. Now with **Urdu language support**!

## Features

* **Voice Activation:** Activate Hansa by saying "Hansa."
* **Multi-Language Support:** Supports both English and Urdu languages
* **Urdu Language Detector:** Automatically detects and processes Urdu text and speech
* **Text Input:** Type your queries and press enter to receive a response.
* **App Control:** Open applications like Notepad and OneNote.
* **Website Navigation:** Open websites like YouTube and Canva.
* **Multimedia Control:** Search and play specific videos on YouTube.
* **Phone and WhatsApp Communication:** Make calls or send messages.

## Installation

### Prerequisites
Make sure you have Python installed. Then install dependencies:

```bash

```

### Setup Instructions

**Clone the Repository:**
```bash
git clone https://github.com/25137126-Shahab/hasna-ai-assistaint.git
cd hasna-ai-assistaint
pip install -r requirements.txt
```

## Usage

### Activating the Assistant
There are several ways to activate Hansa:

- **Voice Activation:** Simply say "Hansa."
- **Urdu Voice Activation:** Say "ہنسا" (Hansa in Urdu)
- **Text Input:** Type your query in the input box (e.g., "How are you?" or "آپ کیسے ہو؟")
- **Keyboard Shortcut:** Press `Window + J` to activate the assistant.

### Supported Commands

#### Query Answering
Ask Hansa questions in English or Urdu, and she'll answer using the Hugging Face API.
**Example:** "Tell me about yourself" / "اپنے بارے میں بتائیں"

#### Opening Applications
- "Open Notepad"
- "Open OneNote"

#### Website Navigation
- "Open YouTube"
- "Open Canva"

#### Multimedia Search
- "Play the video of the 99 names of Allah on YouTube"

#### Phone and Messaging
- "Hansa, make a phone call to Ali Hassan"
- "Hansa, send a message to Ali Hassan"
- "Hansa, make a video call on WhatsApp"

### Future Enhancements

- **Expanded Language Support:** Add support for more languages beyond Urdu and English.
- **Expanded Application Support:** Add support for more desktop and web applications.
